HOMEMADE MASKS

As states begin recommending that everyone wears face masks (in some cases fining those who do not), everyone is looking for cloth masks. We started making my own DIY masks a couple of weeks ago. After testing many patterns, we found the ones that work best. To help others begin making masks we put together this How To Make Homemade Masks!
Each mask takes around 20-25 minutes to make. But we are determined to make as many masks as we can to give to as many people as we can.

Below are the steps for making the beautiful cloth face masks! Hope you enjoy making it!

The beautiful thing about face masks is that there are unlimited possibilities to make new patterns for them, so each one is unique and colorful.

LETS START
MATERIALS

  1. 2 x 100% cotton* size 9 x 7 inches

  2. Elastic 7 inches (x2)

  3. OR strap at least 16 inches (x4)

  4. OR strap at least 16 inches (x2) plus 4.5 inch loop (x2)

  5. *Hospitals are requesting 100% woven cotton for medical caregivers. If you are making for personal use, the higher the cotton percentage the better and the tighter the wave the better.

DIRECTIONS

Place two right sides of the 9×7 fabric together.



The elastic, straps or loops will be placed in each corner with the end sticking out of the fabric and the bulk inside the mask. If using elastic, place one end in the top corner and the other end in the bottom corner along the same short side.
If using four straps, place one end of each strap in each corner. If using two straps, place one end of each of the straps in the top corner and the loop in each of the bottom corners.
Sew around the edge leaving a 2-inch space on one long side (to turn the fabric right side in).
Then turn right-side in.
Make three tucks on each of the short sides of the mask and pin as shown BELOW



Sew all the way around the mask twice to secure everything.
